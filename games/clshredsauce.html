August 7, 2019

Shredsauce turns 7 today! Play the OG version circa 2012.

This is a port from Unity 3.x to Unity 2019.1. I tried my best to keep it as original as possible
Everything was rewritten to C# since Unity's original javascript/unityscript no longer works. I did my best to maintain the original spaghetti structure of the code. Very little of the original code remains in the current version of the game
It now uses WebGL instead of Unity's original web player (which no longer works)
The sliding physics were ripped from the current version since the original implementation no longer works
All of the 3D models had to be reimported
